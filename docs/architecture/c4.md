# Modelo C4 de AURORA

> Nivel 1: Sistema
> Nivel 2: Contenedores
> Nivel 3: Componentes (por contexto)

## Nivel 1: Sistema

```
┌────────────────────────────────────────────────────────────┐
│                                                            │
│                   AURORA SYSTEM                            │
│                 (Intelligence Platform)                    │
│                                                            │
│  - Ingesta datos de entrenamiento                          │
│  - Análisis longitudinal                                   │
│  - Predicción de forma                                     │
│  - Recomendaciones de entrenamiento                        │
│                                                            │
└────────────────────────────────────────────────────────────┘
     ↑                              ↓
     │                              │
┌────┴──────┐                ┌──────┴────────┐
│            │                │               │
│  ATHLETE   │            COACH/TRAINER       │
│ (Web/App)  │            (Optional)          │
│            │                │               │
└────────────┘                └───────────────┘


Integraciones externas (detrás de adaptadores):
- Garmin/Wahoo: Subida de archivos FIT
- Opcional: Strava, Zwift, Apple Health (lectura de datos)
```

## Nivel 3: Componentes (por Bounded Context)

### Ingestion Context

```
┌──────────────────────────────────────────────────────────────────┐
│                    INGESTION CONTEXT                             │
│                                                                  │
│  ┌──────────────┐                                               │
│  │ FIT Upload   │ (HTTP Adapter)                                 │
│  └──────┬───────┘                                               │
│         │ raw bytes                                              │
│  ┌──────┴──────────────┐                                        │
│  │  FIT Parser         │                                        │
│  │ - Read binary FIT   │                                        │
│  │ - Extract records   │                                        │
│  │ - Validate format   │                                        │
│  └──────┬──────────────┘                                        │
│         │ FIT Records                                            │
│  ┌──────┴──────────────────────┐                               │
│  │  Data Converter             │                               │
│  │ - Watts, bpm, cadence, gps  │                               │
│  │ - Timestamp alignment       │                               │
│  │ - Mark gaps                 │                               │
│  └──────┬──────────────────────┘                               │
│         │ Raw Streams                                            │
│  ┌──────┴──────────────────────┐                               │
│  │  Normalizer (DOMAIN)        │                               │
│  │ - Standard units            │                               │
│  │ - Create Stream VOs         │                               │
│  │ - Validate invariants       │                               │
│  └──────┬──────────────────────┘                               │
│         │ NormalizedWorkout                                     │
│  ┌──────┴──────────────────────┐                               │
│  │  WorkoutRepository          │                               │
│  │  (Persistence)              │                               │
│  └──────┬──────────────────────┘                               │
│         │ emit: WorkoutNormalized                               │
│  ┌──────┴──────────────────────┐                               │
│  │  Event Bus                  │                               │
│  │  → Performance listener     │                               │
│  │  → Longitudinal listener    │                               │
│  └─────────────────────────────┘                               │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

### Performance Context

```
┌──────────────────────────────────────────────────────────────────┐
│                  PERFORMANCE CONTEXT                             │
│                                                                  │
│  Listener: WorkoutNormalized event                              │
│         │                                                        │
│  ┌──────┴──────────────────────┐                               │
│  │  WorkoutAnalyzer            │ (DOMAIN SERVICE)              │
│  │ - Detect workout type (Z2,3,4,5,6)                          │
│  │ - Calculate TSS             │                               │
│  │ - Calculate IF              │                               │
│  │ - Analyze variability       │                               │
│  └──────┬──────────────────────┘                               │
│         │ TrainingMetrics                                       │
│  ┌──────┴──────────────────────┐                               │
│  │  WorkoutMetricsRepository   │                               │
│  │  (Persistence)              │                               │
│  └──────┬──────────────────────┘                               │
│         │ emit: WorkoutAnalyzed                                 │
│  ┌──────┴──────────────────────┐                               │
│  │  Event Bus                  │                               │
│  │  → Longitudinal listener    │                               │
│  │  → Coaching listener (opcional)                             │
│  └─────────────────────────────┘                               │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

### Longitudinal Context

```
┌──────────────────────────────────────────────────────────────────┐
│                 LONGITUDINAL CONTEXT                             │
│                                                                  │
│  Listener: WorkoutAnalyzed event                                │
│         │                                                        │
│  ┌──────┴──────────────────────┐                               │
│  │  ProgressAnalyzer           │ (DOMAIN SERVICE)              │
│  │ - Last 12 weeks metrics     │                               │
│  │ - FTP trend                 │                               │
│  │ - Load trend                │                               │
│  │ - Detect improvements       │                               │
│  └──────┬──────────────────────┘                               │
│         │                                                        │
│  ┌──────┴──────────────────────┐                               │
│  │  FormCalculator             │ (DOMAIN SERVICE)              │
│  │ - CTL (chronic training load)                               │
│  │ - ATL (acute training load) │                               │
│  │ - TSB (training stress balance)                             │
│  │ - Form state                │                               │
│  └──────┬──────────────────────┘                               │
│         │ FormState                                             │
│  ┌──────┴──────────────────────┐                               │
│  │  FatigueCalculator          │ (DOMAIN SERVICE)              │
│  │ - Days since hard workout   │                               │
│  │ - Load ratio                │                               │
│  │ - Fatigue state             │                               │
│  └──────┬──────────────────────┘                               │
│         │ FatigueState                                          │
│  ┌──────┴──────────────────────┐                               │
│  │  AthleteFormRepository      │                               │
│  │  (Cache + Persistence)      │                               │
│  └──────┬──────────────────────┘                               │
│         │ emit: FormUpdated                                     │
│  ┌──────┴──────────────────────┐                               │
│  │  Event Bus                  │                               │
│  │  → Prediction listener      │                               │
│  │  → Coaching listener        │                               │
│  └─────────────────────────────┘                               │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

### Prediction Context

```
┌──────────────────────────────────────────────────────────────────┐
│                  PREDICTION CONTEXT                              │
│                                                                  │
│  Listeners: FormUpdated, RaceProfile selected                   │
│         │                                                        │
│  ┌──────┴──────────────────────┐                               │
│  │  PeakFormPredictor          │ (DOMAIN SERVICE)              │
│  │ - Historical form patterns  │                               │
│  │ - Current form state        │                               │
│  │ - Days until race           │                               │
│  │ - Estimate peak window      │                               │
│  └──────┬──────────────────────┘                               │
│         │ CompetitionWindow (with confidence %)                │
│  ┌──────┴──────────────────────┐                               │
│  │  RiskAssessor               │ (DOMAIN SERVICE)              │
│  │ - Fatigue level             │                               │
│  │ - Load balance              │                               │
│  │ - Historical injury patterns│                               │
│  │ - Overtraining risk calc    │                               │
│  └──────┬──────────────────────┘                               │
│         │ OvertrainingRisk                                      │
│  ┌──────┴──────────────────────┐                               │
│  │  PerformanceProjector       │ (DOMAIN SERVICE)              │
│  │ - FTP trend projection      │                               │
│  │ - If you keep this pattern  │                               │
│  │ - Where will you be in 4w?  │                               │
│  └──────┬──────────────────────┘                               │
│         │ PerformanceProjection                                │
│  ┌──────┴──────────────────────┐                               │
│  │  PredictionRepository       │                               │
│  │  (Cache + Persistence)      │                               │
│  └──────┬──────────────────────┘                               │
│         │ emit: PredictionGenerated                             │
│  ┌──────┴──────────────────────┐                               │
│  │  Event Bus                  │                               │
│  │  → Coaching listener        │                               │
│  └─────────────────────────────┘                               │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

### Coaching Context

```
┌──────────────────────────────────────────────────────────────────┐
│                  COACHING CONTEXT                                │
│                                                                  │
│  Listeners: FormUpdated, PredictionGenerated, UI request        │
│         │                                                        │
│  ┌──────┴──────────────────────┐                               │
│  │  InsightGenerator           │ (DOMAIN SERVICE)              │
│  │ - Current form              │                               │
│  │ - Fatigue level             │                               │
│  │ - Overtraining risk         │                               │
│  │ - Days until race           │                               │
│  │ - Generate recommendation   │                               │
│  └──────┬──────────────────────┘                               │
│         │ CoachInsight                                          │
│         │ (action + reasoning + confidence)                     │
│  ┌──────┴──────────────────────┐                               │
│  │  InsightRepository          │                               │
│  │  (Persistence)              │                               │
│  └──────┬──────────────────────┘                               │
│         │ emit: InsightGenerated                                │
│  ┌──────┴──────────────────────┐                               │
│  │  HTTP Response              │                               │
│  │  → UI displays insight      │                               │
│  └─────────────────────────────┘                               │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

```
    │                       │                          │

    │                       │         PredictionGenerated       │
    │                       │                          │       │
    │                       │                          └──────→

```
